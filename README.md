# Maps-V2

A collection of interactive maps for Best Day Ever Venues (BDEV).

## BDEV Booking Reach Map

Interactive map showing where couples live who book each of the six BDEV wedding venues. Colour-coded by venue, with prospective venue locations also marked.

**Venues included:** The Ashes Barns, Foxtail Barns, Whistle Barns, Hanbury Barns, Pendrell Hall, The Mill Barns

**Prospective venues:** Waterworks House (Arnold, Nottingham) and The West Mill (Darley Abbey, Derby)

[Open the map](https://marialarcombe.github.io/Maps-V2/bdev_venue_reach_map.html)
